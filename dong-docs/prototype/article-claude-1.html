dongCoding2026.06.20수정 2026.07.017분👁 1,284

터미널은 오랫동안 차가운 곳이었다. 그런데 그 안에 대화 상대가 생기면, 도구를 쓰는 감각이 통째로 바뀐다.

이 글에서는 설치부터 첫 대화까지를 다룬다. 인라인 수식은 이렇게 — 복잡도 $O(n \log n)$ — 문장 속에 놓인다. 인라인 코드 claude 도 함께.

설치

패키지 매니저는 pnpm을 쓴다.

# 전역 설치 후 첫 실행
pnpm add -g @anthropic-ai/claude-code
claude "이 저장소를 설명해줘"
도구가 말을 걸어오기 시작하면, '어떻게'보다 '무엇을'에 집중하게 된다.

시킬 것과 시키지 말 것

  • 맡겨라: 반복적 리팩터링, 테스트 작성
  • 직접 하라: 되돌리기 어려운 결정, 배포
